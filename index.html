SilaInterview

View on GitHub Download .zip Download .tar.gz

Это jQuery-плагин для создания интерактивных интервью, встраеваемых в страницу.

Интерактивное интервью - это набор вопросов и связанных с ними видео-ответов.

Установка

  • Скачать и разместить файлы silainterview на своем сайте
  • Встроить в страницы HTML-код. Допустим, вы разместили файлы по адресу http://example.com/int , тогда код будет таким:
<script src="http://example.com/int/silainterview.js"></script>
<div id="silainterview1" class="silainterview">
    <h1>Demo interview</h1>
    <div class="sila-description">
        description
    </div>
    <div class="sila-video-player">
        <img src="http://example.com/int/title.jpg">
    </div>
    <div class="sila-questions">
        <h2>Выберите вопрос</h2>
        <ul>
            <li data-video="www.youtube.com/watch?v=-5REkR4b1b0">
                Критерии, которыми руководствовались эксперты при оценке проектов
            </li>
            <li data-video="www.youtube.com/watch?v=SJAF6CPjdbw">
                Что удивило и порадовало экспертов в проектах
            </li>
        </ul>
    </div>
</div>
<script>
    jQuery(function () {
        jQuery('#silainterview1').SilaInterview({
            css: 'http://example.com/int/silainterview.css'
        });
    });
</script>
