SilaInterview

View on GitHub Download .zip Download .tar.gz

Это jQuery-плагин для создания интерактивных интервью, встраеваемых в страницу.

Интерактивное интервью - это набор вопросов и связанных с ними видео-ответов.

Установка

  • Скачать и разместить файлы silainterview на своем сайте
  • Встроить в страницы HTML-код. Допустим, вы разместили файлы по адресу http://example.com/int , тогда код будет таким:
<script src="http://example.com/int/silainterview.js"></script>
<div id="silainterview"></div>
<script>
    jQuery(function(){
        jQuery('#silainterview').SilaInterview({
                        css: 'http://example.com/int/silainterview.css',     
            name: 'interview\'s name',
            descr: 'supa pupa interview',
            photoUrl: 'http://thumbs.dreamstime.com/x/casual-interview-meeting-4232679.jpg',

            questions: [
                {
                    text: '111111111',
                    videoUrl: 'www.youtube.com/watch?v=TStjKm5OiUA'
                },
                {
                    text: '22222222',
                    videoUrl: 'www.youtube.com/watch?v=PtuHeWyDJqQ'
                },
                {
                    text: '3333333',
                    videoUrl: 'www.youtube.com/watch?v=kZ6A6VaWUIU'
                },
                {
                    text: '4444444',
                    videoUrl: 'www.youtube.com/watch?v=DzfUBoD9R7U'
                }
            ]
        })
    });
</script>
